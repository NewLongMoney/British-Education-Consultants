# British Education Consultants

Professional website for British Education Consultants (BEC) - an education advisory firm specialising in British National Curriculum implementation.

## About BEC

British Education Consultants helps schools in Africa, the Middle East, and Asia transition to and deliver the British National Curriculum through:

- **Teacher Training** - Structured programmes for curriculum delivery
- **Leadership Development** - NPQ qualifications and leadership training
- **School Advisory** - Feasibility, accreditation, and quality assurance
- **CPD Programmes** - Continuous professional development

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Instrument Serif, DM Sans)

## Pages

- `index.html` - Homepage
- `about.html` - About Us
- `services.html` - Services
- `contact.html` - Contact

## Development

To run locally:

```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx serve
```

Then open `http://localhost:8000`

## Deployment

This site is deployed on Vercel. Any push to the `main` branch triggers automatic deployment.
